# Installation

UltimateSlice is available for Windows, macOS, and Linux. Choose your operating system and follow the instructions below.

### Windows

1.  Download the **UltimateSlice Setup.exe** from our download page.
2.  Run the installer and follow the on-screen instructions.
3.  Once the installation is complete, launch UltimateSlice from your desktop.

### macOS

1.  Download the **UltimateSlice.dmg** from our download page.
2.  Open the `.dmg` file and drag the **UltimateSlice.app** into your **Applications** folder.
3.  Launch UltimateSlice from your Applications folder.

### Linux

We provide UltimateSlice as a **Flatpak**, **AppImage**, and **deb/rpm** package.

- **Flatpak (Recommended)**:
  ```bash
  flatpak install flathub app.ultimateslice.UltimateSlice
  ```
- **AppImage**: Simply download the `.AppImage` file, make it executable, and run it.
